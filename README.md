Introduction
This repository contains my weekly JavaScript practice as I learn the basics of web development.

I created this project while learning how JavaScript works and how it can be used together with HTML to create interactive websites.

The main purpose of this repository is to practice JavaScript concepts from the beginner level and slowly understand how websites work behind the scenes.

During this learning process, I worked on variables, functions, conditions, loops, DOM manipulation, forms, validation, and asynchronous JavaScript.

Technologies Used
The technologies I used in this project are:
HTML5
JavaScript
Basic CSS
JSON

I mainly focused on learning JavaScript because it is the programming language that makes websites interactive.
